# Contributing to tfcdash

Thank you for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/tfcdash.git`
3. Create a branch: `git checkout -b feature/your-feature-name`

## Development

### Requirements

- Go 1.21+

### Running Locally

```bash
go run main.go
```

### Building

```bash
go build -o tfcdash main.go
```

## Pull Request Guidelines

- Ensure code follows Go conventions (use `go fmt`)
- Describe your changes clearly in the PR
- Test your changes before submitting

## Issues

Feel free to open issues for:
- Bug reports
- Feature requests
- Questions
